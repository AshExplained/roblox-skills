# Roblox Genre Patterns

## Goal

Use proven Roblox genre patterns without copying them blindly.

## Genre Lenses

- Simulator: satisfying repeat action, upgrade ladder, prestige, pets, zones.
- Tycoon: visible build progress, income pacing, co-op or competition, rebirth.
- Obby: immediate challenge, fast retries, checkpoints, cosmetics, UGC-like flex.
- RPG/adventure: quests, zones, combat, loot, bosses, social parties.
- Pet collector: hatch loop, rarity, trading, collection book, events.
- Tower defense: waves, units, upgrades, co-op, map rotation, balance.
- Survival: resource loop, risk/reward, base building, social tension.
- Horror: atmosphere, pursuit, team survival, replayable objectives.
- Battlegrounds: combat readability, matchmaking, ability balance, anti-exploit.
- Roleplay: identity, customization, social spaces, user-generated stories.
- Racing: short sessions, upgrades, ghosts, leaderboards, events.

## Pattern Checklist

For any genre, define:

- First 30 seconds.
- Core loop.
- Progression ladder.
- Social hook.
- Monetization fit.
- Content update cadence.
- Mobile control risks.
- Security-sensitive systems.

## Output

Return:

- Genre fit.
- Differentiating twist.
- MVP feature list.
- Live-ops content pipeline.
- Biggest retention and production risks.
